Código del MCU (ESP32) y frontend web para el invernadero.

# Sistema de Gestion Agro Inteligente

#### 🧠 El proyecto del invernadero automatizado consistirá en una maqueta capaz de monitorear y controlar distintas condiciones de la planta de forma automática. 

#### 🖧 Contará con sensores de temperatura, humedad y presión atmosférica, además de un sistema de riego automático y un buzzer utilizado para espantar animales o insectos. 

#### 🛠️💻 También tendrá una página web desde la cual se podrá visualizar el estado de la planta y supervisar el funcionamiento general del sistema.
